# Detector de Mutantes - Angular Frontend

Este proyecto es una aplicación web interactiva desarrollada en **Angular** que permite validar si una secuencia de ADN pertenece a un mutante o a un humano, basándose en un algoritmo de análisis de matrices de caracteres ($NxN$).

## Características
* **Validación en Tiempo Real:** Interfaz gráfica para ingresar cadenas de ADN separadas por comas o saltos de línea.
* **Visualización de Matriz:** Renderiza dinámicamente la matriz de ADN en una cuadrícula interactiva para facilitar la auditoría visual.
* **Componentes Standalone:** Desarrollado bajo la arquitectura moderna de Angular utilizando componentes independientes y servicios desacoplados para la lógica de negocio.

## Tecnologías Utilizadas
* **Angular** (Versión 17+)
* **TypeScript**
* **CSS3** para el diseño del panel de laboratorio
